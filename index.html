Semester I: Theoretical Preparation

Computer System and Organisation

Q1: What is the role of the CPU in a computer system?
A: Executes instructions by fetch, decode, execute cycles.
Q2: What is the difference between primary and secondary memory?
A: Primary (e.g., RAM) is fast, volatile; secondary (e.g., HDD) is slow, non-volatile.
Q3: Explain the difference between GUI and CUI with examples.
A: GUI (e.g., Windows) uses visuals; CUI (e.g., Linux terminal) uses text.
Q4: Convert the decimal number 25 to binary.
A: 11001.
Q5: What is a NOT gate? Draw its truth table.
A: Inverts input; 0→1, 1→0.
Q6: What are the functions of an operating system?
A: Manages hardware, UI, files, tasks.
Q7: What is cache memory, and why is it used?
A: Fast memory between CPU and RAM for frequent data, reduces access time.
Q8: What is the difference between a compiler and an interpreter?
A: Compiler translates all at once; interpreter line-by-line.
Q9: What are the units of memory in ascending order?
A: Bit, Byte, KB, MB, GB, TB, PB.
Q10: Convert the hexadecimal number 1A to binary.
A: 00011010.
Q11: What is the purpose of De Morgan’s theorem?
A: Simplifies logic, e.g., !(A AND B) = !A OR !B.
Q12: What is the difference between supercomputers and mainframes?
A: Supercomputers for speed; mainframes for reliability.

Programming Fundamentals

Q13: Define an algorithm and list two characteristics.
A: Step-by-step procedure; finiteness, precision.
Q14: What is the difference between high-level and low-level languages?
A: High-level (e.g., C) readable; low-level (e.g., assembly) machine-specific.
Q15: What is time complexity in algorithms?
A: Time as a function of input size, e.g., Big O.
Q16: What is structured programming? Give an example.
A: Uses control structures; e.g., `if-else` in C.
Q17: Explain the purpose of a flowchart.
A: Visually represents algorithm steps.
Q18: What is pseudocode, and why is it useful?
A: High-level algorithm description; aids planning.
Q19: What is the difference between procedural and object-oriented programming?
A: Procedural (e.g., C) procedure-based; OOP (e.g., Java) object-based.
Q20: What does Big O notation represent?
A: Upper bound of time complexity.
Q21: What is a recursive algorithm? Give an example.
A: Calls itself; e.g., Fibonacci.
Q22: How does debugging improve a program?
A: Fixes errors for correct execution.

Introduction to C

Q23: What is the purpose of the `printf` function?
A: Displays formatted output, e.g., `printf("%d", x);`.
Q24: What is the difference between `while` and `do-while` loops?
A: `while` checks first; `do-while` executes once then checks.
Q25: Write a C statement to declare a 2D array of size 3x4.
A: `int arr[3][4];`
Q26: What is recursion in C? Give an example.
A: Function calls itself; e.g., `fact(n) = n * fact(n-1)`.
Q27: What is a pointer in C? How do you declare one?
A: Stores address; `int *ptr;`.
Q28: Write a C snippet to swap two numbers using pointers.
A: `void swap(int *a, int *b) { int temp = *a; *a = *b; *b = temp; }`.
Q29: What is the difference between `&` and `*` operators?
A: `&` gets address; `*` dereferences.
Q30: How do you concatenate two strings without `strcat`?
A: Copy characters manually.
Q31: What is the purpose of the `return` statement?
A: Sends value back, exits function.
Q32: What is the role of `#include`?
A: Imports header files, e.g., `stdio.h`.
Q33: What is the difference between `for` and `while` loops?
A: `for` for known iterations; `while` for condition-based.
Q34: Write a C statement to declare a structure for a student.
A: `struct student { char name[20]; int age; };`.
Q35: What is the significance of `void` in a function?
A: No return value.
Q36: How do you pass an array to a function?
A: Pass array name, e.g., `int arr[]`.
Q37: Write a C snippet to find string length without `strlen`.
A: `int len = 0; while (str[len] != '\0') len++;`.
Q38: What is the difference between pre- and post-increment?
A: Pre (`++i`) increments first; post (`i++`) uses then increments.
Q39: What is a nested function call? Give an example.
A: Function in a function call; e.g., `printf("%d", add(2, 3));`.
Q40: What is the purpose of `scanf`?
A: Reads formatted input, e.g., `scanf("%d", &x);`.

Semester II: Theoretical Preparation

Data Structure

Q41: What is a data structure? Name two types.
A: Organizes data; linear (array), non-linear (tree).
Q42: What is the difference between an array and a linked list?
A: Array fixed, contiguous; linked list dynamic, non-contiguous.
Q43: Explain the push operation in a stack.
A: Adds element to top, increments pointer.
Q44: What is a queue? How does it differ from a stack?
A: FIFO; stack is LIFO.
Q45: Write a pseudocode for linear search.
A: `for i = 0 to n-1: if array[i] == key: return i; return -1;`.
Q46: What is bubble sort? Why is it called so?
A: Swaps adjacent elements; smaller "bubble" to top.
Q47: What is the advantage of a circular linked list?
A: Continuous traversal from any node.
Q48: What is tail recursion? Give an example.
A: Last operation is recursive; e.g., `print(n-1)` last.
Q49: How do you implement a queue using an array?
A: Use `front` and `rear` pointers.
Q50: What is the time complexity of binary search?
A: O(log n).
Q51: What is an abstract data type (ADT)?
A: Defines data/operations, e.g., stack.
Q52: What is the difference between singly and doubly linked lists?
A: Singly one pointer; doubly two.
Q53: Explain the pop operation in a stack.
A: Removes top element, decrements pointer.
Q54: What is a circular queue? How does it improve?
A: Wraps around; reuses space.
Q55: Write a pseudocode for bubble sort.
A: `for i = 0 to n-1: for j = 0 to n-i-1: if arr[j] > arr[j+1]: swap;`.
Q56: What is the worst-case time complexity of linear search?
A: O(n).
Q57: What is the advantage of recursion over iteration?
A: Simplifies code for complex problems.
Q58: How do you detect an empty stack?
A: Top is -1 or NULL.
Q59: What is a priority queue? Give a use case.
A: Prioritizes elements; e.g., task scheduling.
Q60: What is the difference between static and dynamic data structures?
A: Static fixed; dynamic resizable.

Q61: What is the difference between LAN and WAN?
A: LAN small, fast; WAN large, slower.
Q62: What is a protocol in networking? Give an example.
A: Rules for communication; e.g., HTTP.
Q63: Explain the role of a router.
A: Directs data packets between networks.
Q64: What is the difference between simplex and duplex?
A: Simplex one-way; duplex two-way.
Q65: What is the purpose of the OSI model?
A: Standardizes network communication.
Q66: Name the OSI layer for routing.
A: Network layer.
Q67: What is an IP address?
A: Unique device identifier, e.g., 192.168.1.1.
Q68: What is the difference between a hub and a switch?
A: Hub broadcasts; switch targets.
Q69: What is bandwidth in networking?
A: Max data transfer rate (bps).
Q70: What is the difference between TCP and UDP?
A: TCP reliable; UDP faster.
Q71: What is the role of a modem?
A: Converts digital to analog signals.
Q72: What is a star topology? Advantage?
A: Central hub; easy troubleshooting.
Q73: What does HTTP do?
A: Transfers web pages.
Q74: Name the OSI layer for error detection.
A: Data Link layer.
Q75: What is a URL? Example?
A: Web address; e.g., `https://example.com`.
Q76: What is the difference between a repeater and a router?
A: Repeater amplifies; router directs.

Practical Preparation

Q85: What does the `ls` command do in Linux?
A: Lists directory contents.
Q86: Write a C program to check if a number is prime.
A: 
#include 
int main() {
    int n, i, flag = 0;
    printf("Enter a number: ");
    scanf("%d", &n);
    for (i = 2; i <= n / 2; i++) {
        if (n % i == 0) { flag = 1; break; }
    }
    if (flag == 0 && n > 1) printf("%d is prime\n", n);
    else printf("%d is not prime\n", n);
    return 0;
}
Q87: What is the purpose of the `mkdir` command?
A: Creates a new directory.
Q88: Write a C function for factorial using recursion.
A: 
int fact(int n) {
    if (n == 0) return 1;
    return n * fact(n - 1);
}
Q89: What does the `cat` command do?
A: Displays file contents.
Q90: Write a C snippet to reverse a string using pointers.
A: 
void reverse(char *str) {
    char *end = str;
    while (*end) end++;
    end--;
    while (str < end) {
        char temp = *str; *str++ = *end; *end-- = temp;
    }
}
Q91: What does the `mv` command do?
A: Moves/renames files.
Q92: Write a C snippet for sum of first n numbers.
A: 
int sum = 0;
for (int i = 1; i <= n; i++) sum += i;
printf("Sum: %d\n", sum);
Q93: What is the purpose of `rmdir`?
A: Removes empty directories.
Q94: Write a C function to check palindrome.
A: 
int isPalindrome(char *str) {
    char *end = str;
    while (*end) end++;
    end--;
    while (str < end) if (*str++ != *end--) return 0;
    return 1;
}
Q95: What does the `who` command do?
A: Shows logged-in users.
Q96: Write a C snippet to find max in array.
A: 
int max = arr[0];
for (int i = 1; i < n; i++) if (arr[i] > max) max = arr[i];
